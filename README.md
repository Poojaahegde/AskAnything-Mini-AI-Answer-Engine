# AskAnything 🤖
A lightweight AI-powered Q&A engine that uses Wikipedia search and Hugging Face transformers to answer user questions.

## Features
- Retrieves top 3 relevant Wikipedia articles
- Summarizes them using a BART model
- Displays concise, citation-based answers

## Run
```bash
pip install -r requirements.txt
streamlit run main.py
```
